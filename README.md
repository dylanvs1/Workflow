# Claude and Apify Workflow Integration

## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/dylanvs1/Workflow.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Workflow
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
1. Make sure you have your API keys available.
2. Run the workflow:
   ```bash
   node workflow.js
   ```
3. Follow the on-screen instructions to complete the integration process.

For more information, please refer to the project documentation or reach out for support.
